# MVP 기능 체크리스트

작성 기준: 2026-05-25 체크리스트 반영

## 우선순위

- P0: MVP 필수
- P1: MVP 포함 후보, 일정 압박 시 제외 가능
- v2: MVP 이후

## 핵심 결정

- 수업 형태는 그룹 수업과 1:1 수업을 모두 지원한다.
- 수강생 자율예약은 MVP에서 지원하지 않는다.
- 원장이 수업을 만들고 수강생을 초대 또는 배정한다.
- 출석체크는 강사가 한다.
- 수강권은 기간과 회차 결합형이며, 둘 중 먼저 만료되면 종료된다.
- 케어 발송은 MVP에서 제외하고, 신호만 보여준 뒤 원장이 수동 케어한다.
- 강사 정산은 월급 고정 기준이며 자동 정산은 MVP에서 제외한다.
- 클래스노트는 수업 단위로 공유한다.
- 그룹 수업은 같은 클래스노트를 공유하고 개인 코멘트는 MVP에서 제외한다.
- 트레이드 개념은 사용하지 않는다.
- 결석 시 다른 세션 이동 또는 보강으로 처리한다.
- 결석 신청은 수강생이 원하는 결과까지 선택해서 신청하고, 강사가 승인한다.
- 강사는 가능한 시간대를 미리 설정할 수 있어야 한다.
- 게시판은 공지만 제공한다.
- 자료 공유는 클래스노트로 흡수한다.
- 결제 영역은 MVP에서 제외한다.
- 이탈 신호 열람은 원장만 가능하며, 강사 열람은 v2로 둔다.

## P0 범위

### 원장 운영 웹

- 주간/월간 캘린더 뷰
- 수업 등록: 그룹/1:1, 강사, 장소, 시간, 정원
- 반복 일정 등록
- 수업 이동·취소
- 수강생 배정
- 수업 상세: 등록 수강생 명단, 출결 상태, 정원 표시
- 캘린더 정원 표시: 그룹은 `3/5`, 1:1은 수강생 이름
- 일정 충돌 경고: 같은 강사/장소 중복
- 수강생 목록, 검색, 필터
- 수강생 등록: 이름, 연락처, 메모
- 수강생 상세: 수강권, 최근 수업, 출결 이력
- 수강권 상품 등록: 회차, 유효기간, 가격
- 수강생에게 수강권 발급
- 수강권 사용 이력: 차감, 복구 로그
- 수강권 자동 만료
- 수강권 만료 임박 자동 표시
- 위험 수강생 리스트
- 위험 신호 배지: 휴면, 만료임박, 변동잦음, 보강누적, 신규정착중
- 룰 기반 신호 자동 갱신
- 케어 기록: 컨택 이력 메모, 처리 상태
- 강사 등록·초대
- 강사 목록
- 강사별 진행 현황
- 클래스노트 열람 권한 설정
- 공지 작성
- 공지 발송: 전체 또는 특정 클래스 수강생
- 학원 기본 정보
- 브랜딩: 로고, 메인 컬러
- 수강권 만료 정책
- 보강 정책
- 홈 대시보드: 오늘 수업, 이번 주 매출, 등록 수강생 수
- To-do 카드: 결석 신청, 회차 소진 임박, 위험 수강생
- 주간/월간 운영 지표 차트

### Dionomy 회사 웹

- 랜딩
- 데모 신청 폼
- 가격 안내
- CS 게시판: FAQ, 문의 등록, 내 문의 내역

### Dionomy 관리자 세팅 웹

- 학원 목록: 활성, 정지, 해지
- 신규 학원 온보딩
- 화이트라벨 설정
- 빌드 트리거
- 화이트라벨 앱 빌드 상태 모니터
- CS 티켓 큐

### 앱 공통

- 카카오 로그인
- 프로필: 이름, 연락처
- 로그아웃
- 공지 목록·상세

### 수강생 앱

- 홈: 다음 수업 일시, 강사, 장소
- 홈: 잔여 회차, 수강권 만료일
- 내 일정: 배정된 수업 목록
- 수업 상세 보기
- 결석 신청: 사유, 다른 세션 이동 또는 보강 선택
- 결석 신청 상태 보기
- 내 수강권: 회차, 기간, 사용 이력
- 클래스노트 읽기 전용 타임라인

### 강사 모드

- 오늘 수업 목록
- 출석 체크: 참석, 지각, 결석
- 클래스노트 작성
- 클래스노트 이전 기록 보기
- 결석 신청 승인 큐
- 내 수강생 목록
