# H까지 구현 실행 계획

작성일: 2026-05-28

## 진행 방식

각 단계는 같은 루프로 진행한다.

```txt
계획
구현
검증
commit-plan-and-split 커밋 계획
frontend/backend submodule 커밋
root submodule 포인터 커밋
체크리스트 갱신
```

커밋 계획은 항상 다음을 구분한다.

- `frontend` submodule
- `backend` submodule
- root 메타 저장소
- 갱신되는 submodule 포인터
